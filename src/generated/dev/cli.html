$ ricochet -h
ricochet: Put R and Julia into production

Usage: ricochet [COMMAND]

Commands:
  bootstrap       Install R, Julia, or Quarto
  start           Start the ricochet server
  stop            Stop the ricochet server
  upgrade         Upgrade ricochet to the latest version
  secrets-rotate  Rotate server secrets
  help            Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version

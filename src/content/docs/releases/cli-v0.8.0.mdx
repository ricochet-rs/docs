**Released: June 24, 2026**

### Breaking Changes

- **Tidied CLI command lists**: Command organization has been restructured for better usability (#147) (#164)

### Features

- **Dry-run mode for self-updates**: Added `--dry-run` flag to `self update` command to preview updates without applying them (#158)
- **Configuration file support**: App and task settings can now be updated from `_ricochet.toml` configuration file (#159)
- **Default content ID for init**: The `init` subcommand now accepts a default content ID parameter (#156)
- **Instance management commands**: New commands `ricochet app instances {id}` to list instances and `ricochet app stop-instance {id} {pid}` to stop specific instances (#135)
- **Deployment endpoints**: Added support for deployment endpoints (#144)

### Fixes

- **Core compatibility**: Adapted to method renames in ricochet-core ContentType for improved compatibility (#161)

**Released: July 22, 2026**

### Features

- **UI: Actionable error toasts**: Error messages are now persistent, copyable, and include a resync action to help recover from issues more easily.
- **Memory tracking**: The application now monitors memory usage and can identify when crashes are caused by out-of-memory conditions on the native backend.
- **Quarto environment discovery**: Quarto installations are now automatically discovered and displayed in the environment view.

### Fixes

- **UI: Security fix for configuration leak**: Fixed a critical issue where the full server configuration, including the OIDC client_secret, was being exposed to clients.
- **Proxy: Quarto Python sync improvements**: Eliminated false-positive out-of-sync warnings when rerunning Quarto documents with Python.
- **Proxy: Static content deny list**: Implemented proper access controls for static content to prevent unauthorized access.
- **Proxy: Static URL redirects**: Fixed incorrect redirects from `/static/{id}` to `/static/{id}/` to ensure proper resource loading.
- **UI: Kubernetes interpreter display**: Interpreters are now correctly hidden in the UI when running on Kubernetes environments.
- **Rendered task viewing**: The View option now appears for rendered tasks without requiring manual static content configuration.
- **API documentation rendering**: Fixed display issues in Scalar API documentation for Key objects.
- **Frontend stability**: Resolved frontend WebAssembly build inconsistencies by locking Leptos to version 0.8.20.

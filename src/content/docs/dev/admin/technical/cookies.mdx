---
title: "Cookie Usage"
---

Ricochet uses a small number of cookies that are required for the normal operation of the application. These cookies are used exclusively to provide functionality that has been explicitly requested by the user, such as authentication, session management, application routing, and user interface preferences.

## Cookies used by Ricochet

| Purpose                | Description                                                                                                                   | Required |
| ---------------------- | ----------------------------------------------------------------------------------------------------------------------------- | :------: |
| OAuth security         | Protects the OAuth authentication flow by validating authorization requests and preventing cross-site request forgery (CSRF). |    Yes   |
| Authentication session | Persists the authenticated user session between requests.                                                                     |    Yes   |
| Application affinity   | Ensures requests from a user are consistently routed to the same application instance when load balancing is enabled.         |    Yes   |
| Theme preference       | Stores the user's selected interface theme (for example, light or dark mode).                                                 |    Yes   |


## Third-party services

Ricochet does **not** use cookies for advertising, behavioral tracking, cross-site tracking, or marketing purposes.

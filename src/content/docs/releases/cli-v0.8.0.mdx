**Released: June 24, 2026**

### Breaking Changes

- **Tidied CLI command lists**: Command organization has been restructured for better clarity and usability

### Features

- **Dry-run mode for self-update**: Added `--dry-run` flag to the `self update` command to preview updates without applying them
- **Configuration file support**: App and task settings can now be updated directly from `_ricochet.toml`
- **Default content ID for init**: The `init` subcommand now accepts a default content ID parameter
- **Instance management commands**: Added `ricochet app instances {id}` to list app instances and `ricochet app stop-instance {id} {pid}` to stop specific instances
- **Deployment endpoints**: Added support for deployment endpoints to streamline deployment workflows

### Fixes

- **Core compatibility**: Updated to work with the latest ricochet-core ContentType method changes

**Released: October 16, 2025**

### Features

- **Deploy Command**: Added `deploy` command with `--debug` flag
- **Delete Command**: Added `delete` subcommand
- **List Command**: Added `list` command
- **Version Flag**: Added `--version` flag
- **Mac Binaries**: Build Mac binaries in distinct workflow

### Fixes

- **Config Parsing**: Use implicit defaults during config file parsing
- **Config Editing**: Only edit `content.id` and preserve other fields in config.toml

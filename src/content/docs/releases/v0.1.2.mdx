**Released: February 10, 2026**

### Security

- **Database Pool Configuration**: Added explicit SQLite pool configuration to improve connection handling and stability
- **Dependency Updates**: Updated `time` and `git2` Rust crates to address security vulnerabilities

### Fixes

#### Kubernetes

- **Deployment Readiness**: Improved `wait_for_ready` checks and service port handling to ensure more reliable deployment status detection
- **Resource Cleanup**: Deployments are now properly cleaned up when content is deleted, preventing orphaned resources

#### User Interface

- **Theme and Sidebar Preferences**: Fixed cookie path settings to properly persist theme and sidebar preferences across the application
- **Navigation**: Removed slug references in the UI for cleaner navigation

#### General

- **Shell Scripts**: Added execute permissions to shell scripts to prevent execution errors

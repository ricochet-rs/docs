**Released: May 6, 2026**

### Fixes

- **Restore on Kubernetes**: Fixed an issue where restore operations were incorrectly using the PVC name instead of the cache PVC mount path on Kubernetes clusters
- **OS Release Parsing**: Improved robustness of operating system release information parsing in user agent strings and enhanced restore operation diagnostics

**Released: February 16, 2026**

### Features

- **Licensing Support**: Added Keygen.sh licensing with offline verification for license management
- **Configurable Restore Timeout**: Dependency restore job timeout is now configurable to accommodate different project sizes

### Fixes

- **Restore Jobs**: Fixed execution environment image name not being passed correctly to restore jobs
- **Settings Persistence**: Improved settings UI and ensured execution environment field persists correctly when content is updated

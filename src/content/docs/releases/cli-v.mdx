**Released: April 8, 2026**

### Features

- **ShinyPy and RService Support**: Added support for ShinyPy and RService content types during project initialization
- **Quarto Website Detection**: Automatically detects Quarto website projects during initialization
- **Missing Package Warning**: Added warning message when package file is not found after initialization

### Fixes

- **init**: Fixed missing retention field in ContentItem initializer
- **self-update**: Corrected macOS binary name detection in self-update tarball
- **deploy**: Added authentication check before showing initialization dialog

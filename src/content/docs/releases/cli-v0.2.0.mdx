**Released: January 12, 2026**

### Features

- **Config File Init**: Initialize a `_ricochet.toml` file in `ricochet deploy`
- **Task Invocation**: Invoke task items
- **Config Creation**: Create `_ricochet.toml`
- **Install Script**: Added install script

### Fixes

- **Login URL**: Fixed login with trailing slash
- **Deploy Filtering**: Deploy now respects `exclude` and `include` fields of `_ricochet.toml`

**Released: March 3, 2026**

### Features

- **Audit Logging**: Security-related operations are now logged for compliance and monitoring purposes
- **Python Framework Support**: Added content types for Python frameworks to improve compatibility
- **Log Rotation**: Implemented automatic log rotation to manage disk space and maintain system performance
- **OpenTelemetry Integration**: Added observability support through OpenTelemetry for distributed tracing and monitoring

### Fixes

#### Authentication

- **OAuth Error Handling**: OAuth callback errors now redirect users to the login page instead of showing raw 4xx error responses
- **Admin Role Assignment**: Fixed issue where the first user created was not automatically assigned the admin role

#### Kubernetes

- **Multi-Architecture Support**: Resolved issues with multi-arch cluster support and added real-time restore logs for better visibility

#### API

- **HTTP Crate Compatibility**: Fixed version mismatch between HTTP crate and Axum framework

#### General

- **App Overview Hydration**: Resolved panic that occurred during AppOverview hydration
- **Exec Environment Detection**: Fixed detection logic for execution environments in Leptos applications

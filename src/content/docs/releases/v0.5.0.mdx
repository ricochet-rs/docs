**Released: April 2, 2026**

### Features

- **Server-Sent Events**: Added deployment SSE support and improved content SSE functionality
- **Content Thumbnails**: New API endpoint for retrieving content thumbnails, plus thumbnail upload with aspect-ratio-aware rendering
- **User Avatars**: Added support for user avatars with Gravatar/Libravatar integration, custom upload, and OIDC picture sync
- **Public Profiles**: Enhanced explore page cards and added public user profile pages
- **Environment Variables**: Content environment variables are now passed to the restore process

### Improvements

- **Profile Layout**: Improved user profile page layout and design
- **Sidebar**: Reduced expanded sidebar width for better space utilization
- **Log Panels**: Standardized panel width for tasks and application logs

### Fixes

- **Quarto Support**: Fixed Quarto discovery and container bind mount issues
- **Python Environments**: Resolved Python environment restoration problems on hardened systemd services
- **Deployments**: Fixed issue where all running instances would be killed when environment variables were added or deleted
- **Kubernetes**: Corrected deployment name usage when deleting content items
- **Container Isolation**: Added PID namespace and procfs mount to libcontainers for improved isolation
- **Error Handling**: Return proper 400 error code when deployment is missing packages file, plus added .python-version file checking
- **Authentication**: Replaced AuthGate with Leptos protected route for better auth handling

### Security

- **Dependency Update**: Updated Rust tar crate to v0.4.45 to address security vulnerability
